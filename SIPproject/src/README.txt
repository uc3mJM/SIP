INTRODUCCIÓN
El API de mensajes se proporciona para uso del alumno en las prácticas de la segunda parte de SMA.
El API puede ser extendido y modificado por el alumno libremente según su necesidad

DOCUMENTACIÓN
Se proporciona documentación generada con Javadoc para ayudar en la resolución de dudas sobre el API. 
Se ha cumplimentado la información sobre el funcionamiento de los métodos de las clases (los métodos del resto de clases son similares): 
* InviteMessage 
* SDPMessage 
* SIPMessage
* ACKMessage

CODIGO FUENTE
Se proporciona el código fuente para poder ser reutilizado

PROGRAMA DE EJEMPLO
Se proporciona una clase Prueba.java como ejemplo del uso del API
